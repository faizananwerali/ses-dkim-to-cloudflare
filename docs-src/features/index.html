Capabilities

Features at a glance

Zero Upload
Your CSV never leaves your browser. All parsing happens locally using the File API. Nothing is sent to any server.
Domain Detection
The tool reads your domain from the CSV and uses it in the output filename and zone file header comment.
BIND / RFC 1035 Format
Output is a standards-compliant zone file Cloudflare's DNS import accepts directly. TTL is set to 1 (Cloudflare auto TTL).
Theme Support
System, light, and dark mode — persisted to localStorage . No flash on load.
Records Preview
Parsed records shown in a table before download — grouped by type with a count chip.
Smart Filename
Downloaded as cloudflare-dns-import-[domain].txt — no manual renaming needed.
